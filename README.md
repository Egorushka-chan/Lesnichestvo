# Это курсовой проект для 5 семестра по дисциплине "Базы данных"

В проекте реализованы следующие функции:
- Авторизация пользователей
- Просмотр, добавление, редактирование и удаление для всех 7 сущностей
- Количество фильтраций - 2
- Операций поиска - 1
- 1 хранимая процедура

Используемые технологии:
- C# (.NET 8)
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap, jQuery

Для запуска проекта необходимо:
- Выполнить скрипт из файла `database_script.sql` для создания базы данных и заполнения её начальными данными.
- Настроить строку подключения в файле `appsettings.json`.
- Запустить проект из Visual Studio или командной строки.

Как работать с проектом:
- Admin имеет полный доступ ко всем функциям, Manager - ограниченный доступ, Guest - только просмотр данных.
- Для удобства можно использовать следующие логины и пароли:
  - Admin: Логин - "Egor", Пароль - "tobacco"
  - Manager: Логин - "Manager", Пароль - "Manager"
  - Guest: Логин - "Guest", Пароль - "Guest"
